Estilo y Diseño

NUestra Empresa.

Estilo y Diseño se fundo en 2004, cuenta con amplia experiencia en las áreas de diseño y programación web.
El método de trabajo, se fundamenta principalmente en el servicio completo a domicilio. El servicio es totalmente gratuito, el cliente no se ha de desplazar de su lugar de trabajo y no se ha de preocupar de nada.
Ya son más de 400 clientes de todo el territorio nacional los que han confiado en Estilo y Diseño para lanzar su página web corporativa, tienda virtual o portales de contenidos.

1. Reunión inicial

El primer paso en nuestra metodología de trabajo es que el cliente nos indique sus deseos y expectativas, de esta forma tendremos una referencia para establecer con detalle las metas y los objetivos del proyecto. Tras esta reunión se habrá instaurado una base sólida para iniciar el diseño web y desarrollo de su página web.

2. Diseño de página web

El apartado gráfico será estudiado de manera independiente para cada cliente con el objetivo de conseguir una imagen atractiva para su sitio web. Por lo tanto, podemos decir que en esta fase se originan las ideas y con ellas los primeros bocetos del sitio web responsive. Se le ofrecerá varias propuestas de diseños web profesionales que se adaptarán perfectamente a su negocio.

3. Revisión del Diseño Web

Una vez realizados los primeros bocetos del sitio web responsive se le enviarán al cliente para que los revise y confirme su aceptación. Una vez que el cliente esté de acuerdo y nos dé su aprobación se pasará a la siguiente fase del proyecto.

4. Maquetación del sitio.

Se desarrolla el sitio web, cumpliendo las directrices de estandarización W3C. Esta fase nos permitirá la creación web compatible con los navegadores actuales, distintos dispositivos y con contenidos más accesibles para personas con discapacidad.

5. Programación Web

Utilizando código fuente propio y desarrollado con programación web a medida, podremos adaptarnos perfectamente a las más estrictas exigencias de nuestros clientes. Cabe destacar en este apartado los módulos de gestión totalmente personalizables con páginas web creativas y modernas, tiendas on-line, webs corporativas, etc.

6. Testeo del Sitio Web

Durante el periodo de implantación el sitio web será testeado tanto por nuestros programadores como por el cliente, para determinar que se han cumplido todos los objetivos y requisitos establecidos previamente.

7. Formación al cliente.

En las páginas webs que lo exijan, previamente a la puesta online del sitio web, el cliente será formado sobre el manejo del módulo de gestión y administración del mismo. Está formación vendrá acompañada de un manual de referencia y uso.
